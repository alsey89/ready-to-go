# Sensible Full-Stack Template

- NuxtJS

  - tailwind

# BE

- Go
- Comes with
  - echo (server framework)

  - Modules
    - config
    - logs
    - server
    - pg connector

# Dev Environment

- Dockerfiles for each with prod and dev setup
- Docker-compose setup
